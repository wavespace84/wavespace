# 잡코리아 CSS 추출 계획

## 목표
잡코리아(jobkorea.co.kr) 웹사이트의 CSS를 수집하여 디자인 레퍼런스로 활용

## 추출 항목
1. 외부 CSS 파일 (link 태그)
2. 인라인 스타일 (style 태그)
3. 주요 컴포넌트의 계산된 스타일
   - 헤더/네비게이션
   - 버튼 스타일
   - 폼 요소
   - 카드/리스트 컴포넌트
   - 색상 팔레트
   - 타이포그래피

## 저장 구조
```
jobkorea_css/
├── external/          # 외부 CSS 파일들
├── inline/           # 인라인 스타일
├── computed/         # 계산된 스타일 (JSON)
├── screenshots/      # 참고용 스크린샷
└── analysis/         # 분석 결과
```

## 실행 단계
1. Playwright로 브라우저 열기
2. 잡코리아 메인 페이지 로드
3. CSS 리소스 수집
4. 주요 UI 요소 분석
5. 결과 저장 및 정리
