🔔 통일된 토스트 알림 시스템 테스트

시장조사서 페이지의 다운로드 알림 스타일이 전체 프로젝트에 적용되었습니다.
아래 버튼을 클릭하여 다양한 토스트 메시지를 테스트해보세요.

📝 사용 예시 코드

// 성공 메시지
showSuccessMessage('저장되었습니다!');

// 에러 메시지
showErrorMessage('입력값을 확인해주세요.');

// 정보 메시지
showInfoMessage('로그인이 필요합니다.');

// 경고 메시지
showWarningMessage('저장되지 않은 변경사항이 있습니다.');

// 커스텀 시간 설정 (5초)
showToastMessage('5초 동안 표시됩니다.', 'info', 5000);

// 기존 alert 대체
if (window.showSuccessMessage) {
    showSuccessMessage('작업 완료!');
} else {
    alert('작업 완료!');
}
